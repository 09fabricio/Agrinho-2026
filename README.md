# 🌱 Simulador de Estufa Sustentável - Agrinho 2026

> **Tema do Concurso:** Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente.

## 📖 Sobre o Projeto

Este projeto foi desenvolvido como proposta para o concurso **Agrinho 2026**. Trata-se de um **Simulador de Estufa de Morangos** em formato de página web, criado para demonstrar a importância do controle de variáveis ambientais na agricultura moderna. 

O objetivo do simulador é mostrar na prática como o monitoramento preciso da temperatura, da umidade do ar e da umidade do solo pode evitar o desperdício de água e energia, prevenir pragas de forma natural e garantir uma colheita saudável. É a tecnologia aplicada para provar que é possível alcançar o equilíbrio perfeito entre alta produtividade e preservação ambiental.

---

## ✨ Funcionalidades

* **Simulador Interativo:** Painel de controle com ajustes em tempo real para:
  * Temperatura da estufa (°C)
  * Umidade do Ar (%)
  * Umidade do Solo (%)
* **Análise de Sustentabilidade:** O sistema avalia os parâmetros inseridos e gera alertas automáticos. Ele avisa o usuário sobre riscos de desperdício de recursos (como solo encharcado) ou perigos para a planta (como ar muito úmido facilitando fungos).
* **Navegação Dinâmica (SPA):** O site funciona como uma *Single Page Application*, permitindo a transição rápida e fluida entre a área do simulador e as páginas informativas sem precisar recarregar o navegador.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído focado em leveza, rapidez e acessibilidade, utilizando as tecnologias base da web:

* **HTML5:** Para a estruturação semântica de todo o conteúdo.
* **CSS3:** Para a estilização visual, utilizando uma paleta de cores inspirada na natureza e um layout adaptável.
* **JavaScript (Vanilla):** Responsável por toda a inteligência do simulador (cálculos de condições ideais) e pelo funcionamento do menu de navegação.

---

## 🚀 Como Executar o Projeto

Para visualizar e interagir com o site, não é necessária nenhuma instalação complexa. Siga os passos:

1. Faça o download da pasta do projeto.
2. Certifique-se de que os arquivos `index.html`, `style.css` e `script.js`, além da imagem `imagem-agrinho.jpg`, estão salvos no mesmo diretório.
3. Dê um duplo clique no arquivo `index.html`. Ele abrirá automaticamente no seu navegador padrão (Google Chrome, Firefox, Edge, etc.).
4. Utilize o menu superior para navegar e ajuste as barras do simulador para ver os resultados em tempo real!
